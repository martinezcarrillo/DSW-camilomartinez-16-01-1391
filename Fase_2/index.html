Objetivo del sitio web

La finalidad de esta pagina web, es brindar informacion acerca de las media queries y algunos componentes que la integran.

¿Que son las media requieres?

Podemos decir que en desarrollo web, las media queries son módulos CSS3, los cuales permiten la adaptación en representación de contenidos web, en cuanto a las resoluciones de pantalla y equipos digítales, por ejemplo: un Smartphone frente a pantallas de alta definición o la presencia de características de sensibilidad como el braille.

Por otro lado, las reglas media queries “también denominadas MQ" a veces son un tipo de reglas que permiten crear un bloque de código que solo se procesará en los dispositivos que cumplan con los criterios específicos, tales como;

  • @media Screen and [*condicion*]
  • @media Screen and not [*condicion*]

Sintaxis

Las media queries consisten en un tipo de medio adicional y una o más expresiones de características de medios, por lo que varias de estas consultas se pueden combinar utilizando operadores lógicos y tienen una excepción que no distingue entre mayúsculas y minúsculas;

  • Las medias queries se utilizan para aplicar estilos condicionales con las reglas at-@media y @import de CSS.
  • Indicar medios específicos en los elementos Link y otros elementos HTML.
  • Testear y monitorizar los estados de los medios usando los métodos de JavaScript

Operadores Logicos

Los operadores lógicos son empleados para redactar queries utilizando cadenas como not, and y only. Además, se puede combinar múltiples queries en una lista separada, en este caso existe una excepción la cual es; si en cualquiera de los queries de la lista se presenta una condición verdadero en la hoja de estilo asociada se aplicará y por lo cual equivale a una operación lógica "or"

Tipos De Operadores Logicos

Los tipos de operadores logicos son los siguientes:

    Not: Es un operador lógico, el cual se aplica en todo el query y este puede retomar de forma verdadero o falsa, por otro lado, el operador not puede ser usado para negar un query individual y aceptado solo para un query completo.
    And:El operador and es asociado para cumplir múltiples funciones en el desarrollo web y en multimedia, un query básico en este caso requiere de un medio específico para cumplir su función.
    Only: El operador only previene que navegadores antiguos que no soporten queries con funciones complejas apliquen los estilos asignados, los queries son insensibles a las mayúsculas y minúsculas, por otro lado, la media queries que contenga tipos de medio desconocidos siempre lo tomará falso.
